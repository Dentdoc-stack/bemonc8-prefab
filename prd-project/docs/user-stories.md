# User Stories for Flooddashboard

## User Story 1: As a Data Analyst
I want to apply various filters to the data so that I can analyze specific segments and derive insights relevant to my research.

## User Story 2: As a Project Manager
I want to view metrics and KPIs in real-time so that I can assess the project's progress and make informed decisions.

## User Story 3: As a Developer
I want to understand the calculation logic and filters used in the AI studio so that I can implement new features and ensure consistency in data processing.

## User Story 4: As a Stakeholder
I want to receive regular updates on risks and mitigations so that I can be aware of potential issues and their resolutions.

## User Story 5: As a User
I want an intuitive interface that allows me to easily navigate through the dashboard and access the information I need without confusion.
